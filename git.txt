               Git

------------------------------------------------------------------------------

Check git config
    git config -l

Get history overview
    git log --graph --decorate --oneline

Get list of all files in last commit
    git log --name-status HEAD^..HEAD

Tell git to ignore permissions
    git config core.filemode false
